  • Buffalo Showlist, currently in development, and one of my recent undertakings. I have equity in this venture, I’m currently running the technological portion of the site. In order to meet various business requirements and meet possible user expectations, we decided to build a site that would serve an optimal mobile experience, have a flexible and easy to use CMS, a VPS, and fast web server. Those are respectively are Wordpress, EC2, Nginx, acceleration by Varnish, and some help from W3 Total Cache plugin for the cost of near zero dollars (LEMP). The site is fluid and responsive for mobile devices, and utilizes CSS3 and HTML5 features. This is the first project I have used Compass from more control over the CSS. Github is used from version control, and cron jobs run every now and then to backup to the data to S3.
